Provenance Description Ontology (PDO)

Stefanie Rühle (SUB Göttingen)
2017-04-03 (version 2.0)

1. Introduction
---------------
This ontology defines core classes and properties for the description of provenance. 
The ontology was developed for the description of provenance of items from university collections
but is not limited to this context.

2. Namespaces
-------------
The namespace of this ontology is pdo

Following namespaces are used to refer to related ontologies:
@prefix owl: <http://www.w3.org/2002/07/owl#> .
@prefix rdfs: <http://www.w3.org/2000/01/rdf-schema#> .
